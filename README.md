# 🎮 BATTLE PRO - MAX

لعبة معركة 3D احترافية للجوال

## ⚡ طرق التشغيل

### **الطريقة 1️⃣: تثبيت كتطبيق (الأسهل)**
1. افتح الموقع من جوالك
2. اضغط على القائمة (⋮) في Chrome أو (مشاركة) في Safari
3. اختر "تثبيت التطبيق" أو "Add to Home Screen"
4. سيصير لك أيقونة على الشاشة الرئيسية ✨

### **الطريقة 2️⃣: تشغيل في المتصفح**
- افتح الموقع مباشرة من أي متصفح
